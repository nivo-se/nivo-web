# Icons Directory

All icons use the **Lucide React** package.

No exported SVG files needed - icons are imported from npm package.

## Installation

```bash
npm install lucide-react
```

## Usage

```tsx
import { IconName } from 'lucide-react';

<IconName className="w-5 h-5 text-gray-600" />
```
