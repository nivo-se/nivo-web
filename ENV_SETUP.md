## Environment Variables

Add the following entries to your `.env` file(s):

```
# Supabase
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=

# Backend source (local | supabase)
DATABASE_SOURCE=local

# Local database path (optional, defaults to data/nivo_optimized.db)
# LOCAL_DB_PATH=data/nivo_optimized.db

# API Base URL (frontend -> backend)
VITE_API_BASE_URL=

# OpenAI
OPENAI_API_KEY=
OPENAI_MODEL=gpt-4o-mini

# SerpAPI (optional)
SERPAPI_KEY=

# Copper CRM (optional)
COPPER_API_TOKEN=
```
